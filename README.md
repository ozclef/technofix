# Technofix / TESIS-TecnoFix



TECHNOFIX  es la empresa fictisia para mi tesis de Instituto, donde de programación JAVA realizara una app con función de ser un centro privado dedicado a la reparación , venta y compra de servicios, piezas o equipos tecnologicos para apoyar a la comunidad




Proyecto académico y personal desarrollado por **Oscar Cruz Díaz** como parte de la tesis en el Instituto ICO.  
El objetivo es crear un **sistema de marketplace y centro de reparación tecnológica**, combinando:

- **Aplicación Java (TESIS-TecnoFix)**: ejecutable en PC Windows, desarrollado en Eclipse, que simula la gestión de ventas, compras y reparaciones de equipos electrónicos.
- **Frontend Web (Technofix)**: sitio dinámico en HTML/JavaScript desplegado en GitHub Pages y Vercel, que consume datos desde archivos JSON para mostrar un catálogo de productos y servicios.

---

## 🎯 Objetivo
Brindar a la comunidad un espacio donde:
- Se puedan **vender y comprar** equipos tecnológicos usados.
- Se ofrezcan **servicios de reparación** a nivel básico y avanzado.
- Se fomente la **economía circular**, dando valor a dispositivos que aún tienen utilidad.

---

## 🛠️ Tecnologías utilizadas
- **Java (Eclipse IDE)**: para la aplicación de escritorio.
- **HTML, CSS, JavaScript**: para el frontend web.
- **JSON**: como fuente de datos para productos y servicios.
- **GitHub Pages / Vercel**: despliegue del sitio web.

---

## 📂 Estructura del proyecto
- `/TESIS-TecnoFix` → Código Java de la aplicación de escritorio.
- `/technofix/app` → Frontend web con cards dinámicas.
- `items.json` → Archivo de datos con productos y servicios.
- `index.html` → Página principal que renderiza el catálogo.
